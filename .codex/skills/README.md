# Project Skills

This repository includes a local copy of the currently available Codex system skills so agents working in `meal_tracker_app` can discover and use them from within the project.

## Included Skills

- `.system/imagegen`
- `.system/openai-docs`
- `.system/plugin-creator`
- `.system/skill-creator`
- `.system/skill-installer`
- `delegate`
- `gather`

## Project Skills

- `delegate` for splitting work across Codex threads and worktrees.
- `gather` for merging delegated work back into the orchestrator branch.

## Notes

- These skills are available as repository-local references and mirror the Codex environment's system skills.
- Project-specific skills can be added here later if the app grows new workflows.
