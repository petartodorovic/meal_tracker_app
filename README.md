# Macronaut

Eliza Hackathon project: `meal_tracker_app`.

Macronaut is a goal-based nutrition app for logging meals and tracking macros against build, shred, or maintain targets.

## Docs

- `docs/PRD.md`
- `docs/AGENT_BRIEF.md`
- `docs/DATA_MODEL.md`
- `docs/UX_FLOWS.md`
- `docs/IMPLEMENTATION_PLAN.md`

## Current State

This repository contains the v0 app scaffold:

- React + Vite + TypeScript
- Local-first browser storage
- Onboarding with goal mode and editable macro targets
- Today dashboard
- Manual food logging
- Saved foods
- Simple history
- Settings and local reset

## Run Locally

```bash
npm install
npm run dev
```

## Validate

```bash
npm run test
npm run build
```
